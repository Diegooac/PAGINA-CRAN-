PAGINA CRAN
Sistema web de gestión de alumnos y calificaciones con servicios JSON, base de datos y API externa

Descripción:
Sistema web para la gestión de alumnos y calificaciones que permite realizar operaciones CRUD (crear, leer, actualizar y eliminar), utilizando servicios web en PHP, intercambio de datos en formato JSON, validación de formularios y consumo de un servicio externo.

Funcionalidades:
Registro de alumnos
Consulta de alumnos
Edición de datos
Eliminación de registros
Validación de formularios
Servicios que manejan JSON
Conexión a base de datos
Uso de API externa para generación de avatares

Tecnologías utilizadas:
HTML
JavaScript
PHP
MySQL
API Externa
